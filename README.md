# NX Launcher

NX Launcher is an open source launcher that runs **Minecraft: Java Edition** on Android phones and tablets. It brings the desktop Java edition to mobile with a clean interface, a built in mod browser, and the rendering and runtime stack needed to actually play the game.

## Features

- **Play Minecraft Java on Android** — pick a version and launch it, powered by an open source engine with bundled Java runtimes (Java 8, 17, 21 and 25).
- **Version browser** — install any release straight from the official Mojang version list.
- **Mod browser** — search and install mods from both **Modrinth** and **CurseForge** inside the app.
- **Modpacks, resource packs and shader packs** — browse and install without leaving the launcher.
- **Multiplayer** — built in multiplayer support.
- **World and instance management** — manage saves, mods and mod loaders per instance.
- **Renderers and custom drivers** — switch renderers and add a custom GPU driver as an APK or a `.so` file for your device.
- **Accounts** — Microsoft and offline accounts, with custom skin support.
- **Adjustable resolution, in game language and launch options.**

## Building

Every APK is built automatically from source by GitHub Actions. The Android build uses Gradle, the Android SDK and the NDK, and the Java runtimes are bundled into the app.

## Built on

NX Launcher stands on the work of several open source projects:

- **Fold Craft Launcher** by FCL-Team
- **PojavLauncher**
- Most launcher features are based on **HMCLCore**

Please respect their licenses when redistributing this project.
